# zeeframes.com

Full-stack project (client + server).

## Structure
- `client/` — frontend (Vite + React)
- `server/` — backend (Node.js + Express + MongoDB)

## Setup

### Server
```bash
cd server
npm install
# create a .env file with:
#   PORT=...
#   MONGO_URI=...
npm start
```

### Client
```bash
cd client
npm install
npm run dev
```
